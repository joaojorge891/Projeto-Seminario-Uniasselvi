Desenvolvimento Web com Frameworks Modernos

Tweet
Compartilhar
publicado 21/04/2023 15h00, última modificação 05/06/2023 19h35

Descrição
Nos dias de hoje, o desenvolvimento web não se faz mais de forma artesanal, mas sim lança mão de poderosa base provinda através de frameworks que são verdadeiras caixas de ferramentas para aumentar a produtividade dos desenvolvedores com consequentes ganhos para as empresas e cliente finais. Desta forma, a especialização que está sendo proposta busca prover subsídios para que os alunos possam aprender a utilizar esses frameworks projetar construir aplicações web e móveis com a máxima eficiência.

Público Alvo
O curso se destina, preferencialmente, a portadores de diploma de cursos reconhecidos pelo MEC, nas áreas de Desenvolvimento de Sistemas, Processamento de Dados, Engenharia e áreas afins, que desejam atualizar-se e aumentar seus conhecimentos, possibilitando seu ingresso no setor de desenvolvimento Web utilizando frameworks modernos.
